Coloque seus arquivos nesta pasta para que eles possam ser compartilhados no site.

1. Adicione o arquivo dentro da pasta `uploads/`.
2. Atualize o arquivo `data/files.json` para adicionar o título, descrição, categoria, tamanho e URL.
3. Faça commit e push para o GitHub.

Exemplo de entrada em `data/files.json`:

{
  "title": "Meu arquivo importante",
  "description": "Descrição do arquivo para download.",
  "category": "Documentos",
  "size": "2 MB",
  "url": "uploads/meu-arquivo.txt"
}
